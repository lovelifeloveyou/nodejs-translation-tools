## 安装

    npm install yddict -g

## 使用说明

    yd <要查询的单词>

## 自定义设置
  目前自定义设置选项如下，可以通过用户主目录下`config.json`文件来配置

    {
      "proxy":"http://<主机地址>:<端口>" //代理设置
      "color":"<自定义显示颜色>" //自定义字体颜色显示，默认白色
    }
